Frequently Asked Questions

Quick answers to common questions about SF File Manager, security, and support.

Is SF File Manager really secure?

Yes, SF File Manager uses advanced cryptographic algorithms like AES-256 and ChaCha20-Poly1305. It's built on a zero-knowledge principle, meaning only you have access to your encryption keys and data. We are committed to using the latest security protocols to ensure your data is safe.

Can I recover my files if I forget the password?

Due to our zero-knowledge policy, we do not store your passwords or encryption keys. This means if you lose your password, we cannot help you recover your files. We strongly recommend using a secure password manager to store your passwords safely.

How does quantum-resistant encryption work?

Quantum-resistant encryption refers to a new class of cryptographic algorithms designed to be secure against attacks from future quantum computers. While current quantum computers are not yet powerful enough to break existing encryption, our algorithms are a proactive measure to ensure your data is protected for decades to come.

Is the software free to use?

Yes, SF File Manager is an open-source project and is completely free to use. There are no hidden fees or premium subscriptions. We believe that everyone should have access to powerful security tools.
